# Prompts do Copiloto

Este repositório contém prompts personalizados para o meu copiloto técnico, organizados por modos de operação. Cada prompt define o comportamento, stack tecnológica e personalidade do assistente.

## Resumo dos Prompts

### 1. prompt-agent.md
**Modo AGENT CODE**: Focado em transformar requisitos em implementações completas de código. Segue um ciclo de descoberta, planejamento, implementação, verificação e finalização. Utiliza stack Node.js, React.js e Next.js, com personalidade estilo Cortana (calma, confiante e espirituosa).

### 2. prompt-ask.md
**Modo ASK**: Modo de leitura para responder dúvidas, explicar código e diagnosticar erros. Não executa mudanças automaticamente. Mantém a mesma stack e personalidade, priorizando respostas objetivas e orientações curtas.

### 3. prompt-plan.md
**Modo PLAN**: Produz planos de implementação revisáveis antes de qualquer código. Inclui análise de requisitos, diagramas e estruturação de passos. Focado em planejamento seguro, sem execução direta.

### 4. prompt-study.md
**Modo STUDY**: Atua como tutor para explicar conceitos, trade-offs e práticas. Progressão do simples ao avançado, com analogias, exemplos e checkpoints de compreensão. Stack Node.js/TypeScript e React.js/TypeScript.
